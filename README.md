# Smart Parking System for Smart Cities

## 🚗 Overview

Smart Parking System for Smart Cities is an Internet of Things (IoT) based parking management solution designed to improve parking efficiency in urban environments. The system provides real-time parking slot monitoring, online parking reservations, QR-based ticket generation, and parking analytics through an interactive web dashboard.

The project demonstrates how IoT and cloud technologies can be integrated to reduce traffic congestion, optimize parking utilization, and enhance urban mobility in smart cities.

## 🎯 Features

* Real-time parking slot monitoring

## 👥 User Scenarios

### Scenario 1: Parking Slot Reservation

A vehicle owner checks available parking slots, books a slot, receives a QR ticket, and reserves parking before arriving.

### Scenario 2: Parking Administrator Monitoring

Administrators monitor parking occupancy, vehicle movement, slot status, and parking statistics through the dashboard.

### Scenario 3: QR Ticket Verification

Users present QR tickets at the parking facility for booking verification and access authorization.
